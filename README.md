# Crypt
A password manager in your terminal build using python.

## Usage

### Initializing store
```bash
crypt init
```

### Adding account
```bash
crypt add <site-name>
```
**Replace *<site-name>* with the name of the account to use, example: "github"**

### Getting account passphrase
```bash
crypt get <site-name>
```
**Replace *<site-name>* with the name of the account to use, example: "github"**

### Removing an account
```bash
crypt remove <site-name>
```
**Replace *<site-name>* with the name of the account to use, example: "github"**

### List accounts
```bash
crypt list
```
